# Free Software People

We build open-source C++ software — simple, fast, and useful tools for everyday computing.

---

## What We Do

Our focus is on creating lightweight, efficient applications written in modern C++.  
We build tools we actually need — and share them with everyone.

So far, we've developed:

- Text Editor – minimal, fast, distraction-free writing
- Code Editor – built for developers who value speed and control
- Media Player – lightweight playback with no unnecessary bloat

More projects are always in the works.

---

## Philosophy

We believe software should be:

- Fast – native performance matters
- Simple – no unnecessary complexity
- Open – transparent and community-driven
- Practical – built to solve real problems
